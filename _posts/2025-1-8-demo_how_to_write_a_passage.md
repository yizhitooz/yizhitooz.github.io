### Git 设置

#### 用户名和邮箱

打开 Git Bash / 终端，输入指令：

```bash
git config --global user.name 'your_username' # 你的 Gitee 用户名
git config --global user.email 'your@mail.com' # 你的 Gitee 绑定的邮箱
```

通过上诉命令的设置，用户做出的变动才能被 Gitee 平台识别。
